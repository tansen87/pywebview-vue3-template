# pywebview vue3 template

> Quickly building Python desktop applications

### 运行环境
- Node.js 16.14+
- pnpm 8.x+
- Python3.8-3.11

### 初始化
```shell
pnpm run init
```

### dev
```shell
pnpm run start
```

### build
```shell
# no console window
pnpm run build

# with console window
pnpm run pre

# with folder
pnpm run build:folder
```
