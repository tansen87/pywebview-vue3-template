# pywebview-vue-template

> Quickly building Python desktop applications

### Operating environment
- Node.js 18+
- pnpm 8.x+
- Python3.8-3.11

### screenshot
![homepage](/demo/homepage.png)

### install
```shell
pnpm run init
```

### dev
```shell
pnpm run start
```

### build
```shell
# no console window
pnpm run build

# with console window
pnpm run pre

# with folder
pnpm run build:folder
```
