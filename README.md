### 运行环境

- Node.js 16.14+ ([Node.js 安装教程](https://blog.pangao.vip/NodeJs安装教程/))

- pnpm 8.x+ ([pnpm 安装教程](https://pnpm.io/zh/installation))

- Python3.8-3.11 ([Python 安装教程](https://blog.pangao.vip/Python环境搭建及模块安装))

### 初始化

```shell
pnpm run init
```

### dev

```shell
pnpm run start
```

### build

```shell
# no console window
pnpm run build

# with console window
pnpm run pre
```
